## 投稿2（解説）

【保存推奨】

Claude Code v2.1.201で
権限まわりの挙動が変更された。

デフォルトの動作が「Manual」モードに切り替わっている。

ポイントはここ👇

▶︎ `AskUserQuestion`ダイアログが自動続行しなくなった（/configでタイムアウト設定は可能）
▶︎ CLI・VS Code・JetBrains全てでデフォルトが「Manual」に統一
▶︎ `--permission-mode manual` や `"defaultMode": "manual"` も同様に反映

つまり、

意図しない自動実行が減って
安全に使いやすい方向に調整された変更ですよね✨

アップデート内容はこちら👇
https://github.com/anthropics/claude-code/releases/tag/v2.1.201
